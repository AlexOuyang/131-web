# 131-web

Public course materials for [UCSD CSE 131](https://ucsd-progsys.github.io/131-web)

## Update

Either do 

```bash
make upload
```

or, if you prefer

```bash
make 
cp -r _site/* docs/ 
git commit -a -m "update webpage"
git push origin master
```
